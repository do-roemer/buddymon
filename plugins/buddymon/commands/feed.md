# Buddymon Feed

Feed your buddymon with tokens burned from Claude Code sessions. Unclaimed sessions are converted into XP, leveling up your buddy and increasing their stats.

## Instructions

```bash
cd ${CLAUDE_PLUGIN_ROOT}/../.. && npm run buddymon -- feed
```
