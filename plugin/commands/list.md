# Buddymon List

Display all buddymon fighters currently in the arena with their basic info.

## Usage

```bash
npm run buddymon -- list
```

## What You'll See

- **Buddy Name**: The fighter's name
- **Terminal Tamer**: The human behind the buddy
- **Species**: The type of buddymon (duck, octopus, robot, etc.)
- **Class**: The fighting class (explorer, builder, commander, architect, debugger)
- **Level**: Current level based on activity
- **Record**: Win/loss record

## Example Output

```
Name          Tamer       Species  Class      Level  Record
Rindwick      Dominik     Capybara Explorer   12     5-2
DataDragon    Alex        Dragon   Architect  15     8-1
...
```
