# Buddymon Stats

Display a detailed breakdown of how your fighter card stats are calculated from your Claude Code usage.

## Usage

```bash
npm run buddymon -- stats [--tamer "Your Name"]
```

## What You'll See

- **Power**: Based on code complexity and task difficulty
- **Speed**: Calculated from session frequency and responsiveness
- **Defense**: Reflects code stability and error handling
- **Special**: Related to unique tool usage and creativity
- **HP**: Derived from total sessions and engagement duration

## Examples

```bash
npm run buddymon -- stats
npm run buddymon -- stats --tamer "Your Name"
```

This command helps you understand which coding activities boost which stats, so you can optimize your fighter!
