# Buddymon Leaderboard

Display the current leaderboard rankings for all buddymon fighters in the arena, sorted by win rate and record.

## Usage

```bash
npm run buddymon -- leaderboard
```

## What You'll See

Top fighters ranked by:
- **Wins**: Total number of battles won
- **Losses**: Total number of battles lost
- **Win Rate**: Win percentage
- **Rank**: Position on the leaderboard

## Example Output

```
Rank  Name           Tamer      Wins  Losses  Win Rate
1.    DataDragon     Alex       8     1       88.9%
2.    Rindwick       Dominik    5     2       71.4%
3.    CodePhoenix    Sam        4     3       57.1%
...
```
